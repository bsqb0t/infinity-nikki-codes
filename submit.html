提交搭配码

填写信息，生成提交文本，然后到 GitHub 粘贴提交

提交文本已生成！按以下 3 步操作：

1 复制提交文本
点击下方按钮复制，这段文本包含了你的搭配码全部信息。
2 到 GitHub 提交 Issue
  1. 点击下方按钮打开提交页面
  2. Title（标题）填写：[搭配码] 搭配名称
  3. 正文区域 粘贴刚才复制的文本
  4. 如有图片，直接把图片文件 拖入 正文编辑区域
  5. 点 Submit new issue 提交
打开 GitHub 提交页面
3 等待自动处理
提交后系统会自动：
• 创建 Pull Request（合并请求）
• 关闭你的 Issue
• 维护者审核 merge 后网站自动更新（约 30 秒）

编辑 Issue 会自动同步更新 PR，如果需要修改搭配信息，直接编辑你提交的 Issue 即可。
